# schrodingerbox
Website to fins educacional content.
Using this repository to host the site to calculate grades (to help my classmates) :P

# Main page
The initial page have the option to read some math content and to calcultae grades. Acording to the brazilian educational systemy, our activies are avaliate for 0 to 10. Them the teacher make the media of all the activies.
Example: 9 + 8 + 9 = 26. 26/3= 8,6. This will be the final grade of the unity. 

# Options
You can choose if your media is 6, 7 or (in some schools) 5. You also can edit, delete and add new notes to calculate.
